# Data

Download the Diamonds dataset from Kaggle and place `diamonds.csv` in this folder:

https://www.kaggle.com/datasets/shivam2503/diamonds

Or with the Kaggle CLI configured:

```bash
kaggle datasets download -d shivam2503/diamonds -p . --unzip
```

Expected: ~53,940 rows, columns `carat, cut, color, clarity, depth, table, price, x, y, z`
(plus a leftover index column, which `src/data_prep.py` drops).

`diamonds_clean.csv` is generated by the pipeline — do not edit it by hand.
